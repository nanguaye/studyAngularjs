比如我们要建一个项目,写好后上传到github仓库,也可以从github上拉取仓库里的项目.

1.在github上创建一个新的项目仓库,(--没账号的自行百度...)如下图:

PS:1.git命令的时候,我们可以自定义一些方法:
比如:,git pull --rebase 我们可以定义成 gup
git stash,git pull --rebase,git stash pop 我们可以定义为gsup
1.vi ~/.zshrc 打开配文件,然后 i 键进入编写模式
2.gup='git pull --rebase'
   gsup='git stash && gup && git stash pop'
3.定义好后,按esc 退出编辑模式,然后按 : 输入wq //保存并退出的意思.
